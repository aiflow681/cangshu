【游戏名称】
仓鼠照顾游戏

【游戏介绍】
这是一款H5原生开发的宠物养成类小游戏。游戏中，你需要照顾3只可爱的仓鼠，满足它们的饮食、饮水和娱乐需求。仓鼠们会在多个笼子间自由活动，你要做的就是让它们保持健康快乐！

【玩法说明】
游戏中有3只不同颜色的仓鼠（棕色、白色、灰色），它们会在5个相互连通的笼子中自由活动。

仓鼠需求：
仓鼠有4项需求指标，会随时间自动变化。
1. 饥饿 - 会逐渐增加，需要及时喂食
2. 口渴 - 会逐渐增加，需要及时加水
3. 精力 - 会逐渐消耗，仓鼠睡觉或玩跑轮可恢复
4. 快乐 - 受其他需求影响，满足需求和抚摸可提升

互动道具：
笼子里有多种互动设施供仓鼠使用。
1. 食物碗（蓝色碗）- 点击添加食物，仓鼠饿了会自动去吃
2. 水瓶（蓝色瓶）- 点击加水，仓鼠渴了会自动去喝
3. 跑轮（红色轮）- 仓鼠会自动去玩，可以恢复精力和增加快乐
4. 隧道（绿色管）- 仓鼠会躲进去休息，增加快乐值

操作：
你可以通过鼠标或触摸屏幕进行互动。
1. 点击食物碗 - 添加食物
2. 点击水瓶 - 加水
3. 点击仓鼠 - 抚摸增加快乐值
4. 空格键 - 暂停/继续游戏
5. R键 - 重置游戏
6. 点击"自定义"按钮 - 更改仓鼠和笼子颜色

目标：
保持仓鼠的各项需求在健康范围内，让它们快乐生活！

Tips：
• 屏幕左上角显示第一只仓鼠的状态条
• 仓鼠会根据需求自动行动（吃、喝、玩、睡）
• 食物碗和水瓶旁会显示剩余百分比
• 仓鼠可以通过管道在不同笼子间移动
• 鼠标悬停在互动物品上会显示指针

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用了Canvas 2D渲染技术，采用面向对象编程（OOP）设计模式，实现了AI行为状态机、物理碰撞检测、响应式布局等技术。
