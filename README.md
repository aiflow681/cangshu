【游戏名称】帮我看一下仓鼠

【游戏介绍】
这是一款H5原生开发的休闲模拟类小游戏。游戏通过嵌入的方式提供了一个虚拟仓鼠模拟器，你可以观察和与可爱的虚拟仓鼠互动。游戏模拟了真实仓鼠的生活习性，仓鼠会在笼子里自由活动、吃饭、睡觉和玩耍，为用户提供放松和治愈的体验。

【玩法说明】
操作：
- 使用鼠标滚轮缩放视角，观察仓鼠的不同细节
- 按住小仓鼠可以将其拖动到笼子中的任意地方
- 观察仓鼠的自然行为和活动
- 仓鼠会根据其内在需求自主行动（吃饭、睡觉、玩耍等）

规则：
- 仓鼠会自主进行各种活动
- 仓鼠的互动需要适度，避免过度干扰
- 观察仓鼠的日常生活习性
- 仓鼠会自动回到其认为合适的位置

目标：
- 观察和了解仓鼠的生活习性
- 体验与虚拟宠物互动的乐趣
- 享受轻松治愈的游戏时光
- 尝试将仓鼠移动到不同位置，观察其反应

Tips：
- 适度缩放视角，观察仓鼠的细节动作
- 尝试将仓鼠放到不同的地方，看它如何适应新环境
- 观察仓鼠的日常行为模式，了解它的生活习性
- 给仓鼠一些时间适应，不要过度打扰
- 观察仓鼠对不同环境的反应和适应能力

技术介绍：
- 通过嵌入外部网站（Hammy Hamster）实现仓鼠模拟功能
- 响应式设计，适配各种屏幕尺寸
- 流畅的动画效果，模拟真实仓鼠的移动和活动
- 物理引擎模拟仓鼠的运动和互动
- 简单的用户交互界面，提供直观操作体验
